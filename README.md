# To-Do App

A simple and elegant task management application built with HTML, CSS, and JavaScript. Helps users organize, track, and manage their daily tasks with an intuitive and responsive user interface.

## Features
- ✅ Add, edit, and delete tasks
- ✅ Mark tasks as complete
- ✅ Task persistence with local storage
- ✅ Clean and responsive design
- ✅ Easy-to-use, lightweight interface

## Demo
This is a single-page application perfect for quick task management and daily planning.

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started
1. Clone the repository
2. Open `index.html` in your web browser
3. Start managing your tasks!
